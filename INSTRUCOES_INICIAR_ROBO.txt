INSTRUCOES PARA INICIAR O PROJETO MEU ROBO

1. Abra o terminal.

2. Entre na pasta do projeto:

cd /mnt/arquivos/projetoMeuRobo

3. Ative o ambiente virtual:

source .venv/bin/activate

4. Inicie o sistema:

meu-robo-web

5. O sistema vai abrir automaticamente no Google Chrome.

Endereco local da interface:

http://127.0.0.1:8000

6. Para usar o robo do LinkedIn:

- Cadastre pelo menos um titulo de busca.
- Cadastre pelo menos uma localidade.
- Clique no botao "Iniciar coleta LinkedIn".

7. Para usar o robo do Indeed:

- Cadastre pelo menos um titulo de busca.
- Cadastre pelo menos uma localidade.
- Clique no botao "Iniciar coleta Indeed".
- Considere essa coleta experimental, pois o Indeed pode bloquear navegadores automatizados.

8. O robo vai abrir um Chromium separado e navegar para a plataforma escolhida.

9. Se o LinkedIn pedir login, 2FA, captcha ou verificacao manual, resolva no Chromium aberto.
O robo aguardara ate 10 minutos.

10. Para cada vaga encontrada, o robo tentara abrir a pagina da vaga e salvar no banco:

- titulo extraido
- empresa
- localidade extraida
- descricao da vaga
- data da extracao

11. Para configurar usuario e senha do LinkedIn, edite o arquivo:

/mnt/arquivos/projetoMeuRobo/.env

E preencha:

LINKEDIN_EMAIL=seu_email_linkedin
LINKEDIN_PASSWORD=sua_senha_linkedin

12. Para usar a avaliacao de vagas com IA:

- Guarde os arquivos privados em:

/mnt/arquivos/projetoMeuRobo/private/agent_context/agenteRh.md
/mnt/arquivos/projetoMeuRobo/private/agent_context/cv.md

- Por padrao, a avaliacao usa o Ollama local. Instale o modelo e mantenha o
  servico do Ollama ativo:

ollama pull gemma3:4b

- A configuracao padrao equivale a:

AI_PROVIDER=ollama
OLLAMA_URL=http://127.0.0.1:11434
OLLAMA_MODEL=gemma3:4b
OLLAMA_NUM_CTX=16384

- Para usar a OpenAI no lugar do Ollama, configure no arquivo .env:

AI_PROVIDER=openai
OPENAI_API_KEY=sua_chave_openai
OPENAI_MODEL=gpt-5.5

- Na tela "Vagas", clique em "Avaliar vagas pendentes".

Tambem e possivel rodar pelo terminal:

meu-robo-avaliar-vagas --limit 20

13. Para parar o sistema:

No terminal onde o sistema esta rodando, pressione:

CTRL + C
